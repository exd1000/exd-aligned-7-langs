| Language     | Native String Formatting Style                  |
|--------------|--------------------------------------------------|
| Python       | `f"Hi {name}"`                                   |
| TypeScript   | `` `Hi ${name}` ``                               |
| Rust         | `format!("Hi {}", name)`                         |
| Go           | `fmt.Sprintf("Hi %s", name)`                     |
| Julia        | `"Hi $name"`                                     |
| **Haskell**  | `"Hi " ++ name ++ "!"`                           |
| **Clojure**  | `(str "Hi " name "!")` or `(format "...", ...)`  |
